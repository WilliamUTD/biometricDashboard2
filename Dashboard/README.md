## Dashboard Module
# Prerquisites
Python 3.X
Cognionics data stream

# Setup
After installing python, these packages must be installed as well.
- PyQt5
- pyqtgraph
- matplotlib
- numpy
- pylsl
- scipy

to install these packages use:
- Windows
```
python -m pip install "package name"
```
- Mac OS
```
pip install "package name"
```
